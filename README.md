## 注意事项

- 该脚本使用 [zsh](https://ohmyz.sh) 语法，与 `bash` 语法有区别，后续我会补上 `bash` 版本的脚本。
- 使用该脚本需要先安装 [ffmpeg](https:www.ffmpeg.org) 工具（推荐使用 [homebrew](https://brew.sh) 安装，简单省事~）

## 使用方式

终端中输入下面的命令即可执行

```bash
zsh ffmpegs.zsh 目录 类型
```
